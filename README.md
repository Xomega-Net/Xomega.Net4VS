# Xomega.Net for Visual Studio

[Xomega.Net for Visual Studio](https://xomega.net/docs/visual-studio/overview) extension allows you to quickly create, model and generate data-driven .Net applications using reusable layered architecture and our open-source [Xomega Framework]().

The current repository hosts issues, questions, discussions and feature requests for the [Xomega.Net for Visual Studio](https://xomega.net/docs/visual-studio/overview) extension.
