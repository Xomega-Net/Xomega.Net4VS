# Xomega platform community

The current repository hosts issues, questions, discussions and feature requests for the Xomega.Net low-code development platform.

## What is Xomega.Net platform

Xomega.Net is a low-code platform tageted for .NET developers that allows you to quickly create, model and generate data-driven .NET web or desktop applications.

The platform consists of the following components:

- [Visual Studio extension](https://xomega.net/docs/visual-studio/overview) with a solution wizard for your target architecture(s), and a special project for modeling your application.
- [Xomega Generators](https://xomega.net/docs/generators/overview) for generating (and regenerating!) all application layers and UI views using architecture best practices.
- Open-source [Xomega Framework](https://github.com/Xomega-Net/XomegaFramework) that powers the generated application, and helps you to write clean, reusable and maintainable code.

## How to install Xomega.Net

You can [download Xomega.Net for Visual Studio](https://xomega.net/System/Download.aspx) extension for your version of Visual Studio, and install it locally.

To use the product, you can [order a free Basic or Evaluation license](https://xomega.net/System/Order.aspx) or purchase a Full license.

## Questions and Discussions

This repository allows you to browse or ask questions, share ideas and otherwise engage with the Xomega.Net developers and community members on our [Discussions](../../discussions) page. We love your feedback, and this is a great place to **get involved**.

You will be notified about the discussions that you participate in, those where you get mentioned or the ones that you would like to watch.

## Issues and Feature Requests

Some questions or ideas may get converted to issues or feature requests that are tracked on our [Issues](../../issues) page. This will allow us to prioritize them, put them on our road map, and ultimately include them in a future release.
