# Xomega.Net for Visual Studio

[Xomega.Net for Visual Studio](https://xomega.net/docs/visual-studio/overview) extension allows you to quickly create, model and generate data-driven .Net applications using reusable layered architecture and our open-source [Xomega Framework](https://github.com/Xomega-Net/XomegaFramework).

If you don't have Xomega.Net for Visual Studio installed yet, you can [download it here](https://xomega.net/System/Download.aspx).

The current repository hosts issues, questions, discussions and feature requests for the this Visual Studio extension.
